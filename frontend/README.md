# Frontend (Next.js 14 + Tailwind)

Next.js 14 application using the App Router, TypeScript, Tailwind CSS, ESLint, and Prettier.

## Requirements

- Node.js 18.17+
- npm (or pnpm/yarn)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Quality

```bash
npm run format   # Prettier
npm run lint     # ESLint
npm run typecheck
```
